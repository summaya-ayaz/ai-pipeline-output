# Counter App

Build a single-page web app with the following acceptance criteria:

- The page displays a numeric counter, initially `0`, inside an element with `data-testid="counter-value"`.
- A button with `data-testid="increment-btn"` increases the counter by 1 on click.
- A button with `data-testid="decrement-btn"` decreases the counter by 1 on click.
- A button with `data-testid="reset-btn"` sets the counter back to 0.
- The counter value persists across page reloads via `localStorage` under the key `counter`.
- The page has a visible `<h1>` heading containing the text "Counter".
